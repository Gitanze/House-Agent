# 标准住宅镜头库

将下表作为基础模板。先按户型删减和改名，再根据真实连接关系调整起点与方向。

| 景别 | 空间 | 运镜 |
|---|---|---|
| 全景 | 客厅 | 从入户门进入，稳定前推，逐步呈现客厅全貌 |
| 全景 | 客厅 | 从玄关点位向窗缓慢推进，展现客厅进深与落地窗 |
| 全景 | 客厅 | 从不同机位横摇，补充呈现客厅布局与空间关系 |
| 全景 | 餐厅 | 客餐厅分开时单独拍摄，从入口推进并以多角度横摇展现餐厅全貌 |
| 全景 | 厨房 | 从厨房入口建立空间，呈现厨房全景 |
| 全景 | 厨房 | 从不同机位横摇、缓推或竖摇，展现操作台、橱柜与厨房全貌 |
| 全景 | 主卧 | 从主卧入口推进，呈现主卧全貌 |
| 全景 | 主卧 | 从不同机位横摇，补充展现主卧布局 |
| 全景 | 主卧 | 沿真实动线推进，连续呈现主卧与相连的主卫、衣帽间 |
| 全景 | 衣帽间 | 从主卧向衣帽间推进，呈现衣帽间全貌 |
| 全景 | 衣帽间 | 从不同机位横摇或竖摇，展现柜体布局与纵向收纳 |
| 全景 | 次卧1 | 从走廊向次卧推进，呈现次卧1全貌 |
| 全景 | 次卧1 | 从不同机位横摇，补充展现次卧1布局 |
| 全景 | 次卧3/书房 | 从走廊向房间推进，呈现次卧3/书房全貌 |
| 全景 | 次卧3/书房 | 从不同机位横摇，补充展现次卧3/书房布局 |
| 全景 | 公卫1 | 从走廊向公卫推进，呈现公卫全貌 |
| 全景 | 公卫1 | 从不同机位横摇，补充展现公卫布局 |

## 扩展模板

遇到标准库未覆盖的常见空间时，沿用以下句式：

| 景别 | 空间 | 运镜 |
|---|---|---|
| 全景 | 玄关 | 从入户门缓慢推进，呈现玄关尺度、收纳与通往公共区的关系 |
| 全景 | 阳台 | 从室内向阳台推进，呈现室内外连接、采光与景观 |
| 全景 | 次卧N | 从走廊向次卧N推进，呈现房间全貌 |
| 全景 | 次卧N | 从不同机位横摇，补充展现房间布局 |
| 全景 | 公卫N | 从走廊或公共区向公卫N推进，呈现公卫全貌 |
| 全景 | 公卫N | 从不同机位横摇或竖摇，展现干湿分区与主要设施 |
| 全景 | 储物间 | 从入口缓慢推进，并以横摇或竖摇展现内部收纳 |
| 全景 | 多功能房 | 从入口推进建立空间，再从不同机位横摇展现功能布局 |

## 适配原则

- 没有落地窗时，将“落地窗”改为“主要采光窗”，不得照搬。
- 空间狭窄无法推进时，改用入口定机位横摇或小幅后拉。
- 厨房为开放式时，优先增加从餐厅向厨房推进的关系镜头。
- 卫生间镜面较多时，选择避开摄影师倒影的斜向机位。
- 房间编号缺失或不连续时，忠实采用项目原始命名。
