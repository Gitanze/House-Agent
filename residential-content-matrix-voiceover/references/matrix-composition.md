# 矩阵组合规则

## 组合顺序

1. 目标客户决定“说给谁听”。
2. 口吻决定“谁在说、怎么说”。
3. 重点决定“这条视频怎么组织”。
4. 人工亮点决定“具体展开到哪些细节”。

## 人工亮点展开

- 逐条审计人工亮点，不遗漏。
- 不把亮点集中罗列，要放进相关空间、人物动作或生活场景。
- 如果亮点包含学校、配套、通勤，应作为目标客户选择房子的原因。
- 如果亮点包含装修、改造、收纳，应作为日常顺手的证据。
- 如果亮点包含某个空间，应优先让该空间成为核心段落。

## 抽象口吻边界

- 抽象口吻要敢写，允许强梗和极致比喻。
- 事实不能飘：不存在的房间、学校、改造、配套一律不写。
- 家庭客户也可以抽象，但梗要围绕孩子、收纳、学习、厨房、动线等真实家庭场景。
- 夸张句后面必须接一个真实空间或真实亮点，让观众笑完知道房子到底好在哪。

## 输出策略

- 先给一句核心命题。
- 再按口吻进入故事。
- 中段让人工亮点自然出现。
- 结尾抛出一个与目标客户和重点相关的问题，不做硬销售。
