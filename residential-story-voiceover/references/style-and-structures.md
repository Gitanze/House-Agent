# 房产故事线风格与结构

## 共同语言风格

### 基础分类
- 100平以下为小户型，主要按照单身人士、年轻家庭为主要目标群体，凸显出户型虽小，但是好用。
- 100-200平为中等户型，主要按照一家三口，三代共居为主要目标群体，凸显出性价比，居住体验好，改善型住宅，周边配套好。
- 200平以上为大户型，主要按照一家三口，三代共居为主要目标群体，凸显出房屋格局好，大气，良好的居住体验，重点关注卫生间数量，若卫生间数量大于2则可关注其所在位置，强调每个卧室都有自己的生活，可改造性强。

### 注意要点
- 不要出现卧室A、卧室B这类结合平面图才能够理解的说法，换成“这边的卧室”或“南向的卧室”等口语化表达。


### 语气

- 像朋友分享一次真实看房或居住体验。
- 先讲“我为什么在意”，再讲房子有什么。
- 允许轻微犹豫、自我修正和口语化插入，使表达像现场产生。
- 评价克制。用“确实少见”“短期内不用动”“有点心动又有点纠结”替代绝对判断。

### 断句

把信息拆成可呼吸的口播单位，根据空间亮点分布详略得当地讲述每间房间特点，每个空间1-3句讲解词：

```text
这套房子，我原本以为窗景是核心看点。
结果真正让我停下来的，
是阳台旁边那扇不起眼的小门。
```

不要写成密集说明书：

```text
该房源拥有宽敞明亮的客厅、完善的收纳系统以及高品质装修。
```

### 生活化表达

优先写：

- 回家以后先做什么。
- 一件东西具体放在哪里。
- 入住第一周、下班后或周末会发生什么。
- 这个设计少掉了哪一种麻烦。
- 当前舒服与未来变化之间有什么冲突。

避免只写：

- 空间宽敞，功能齐全。
- 装修品质很高。
- 收纳能力强。
- 居住体验很好。

### 高赞生活流结构

这是一种通用叙事结构，只学习组织方式。每次使用时，都必须替换成当前故事线的叙述身份、时间状态和房源事实。

结构顺序：

1. 日常状态开场：先给一个正在发生的生活状态，如刚下班、准备做饭、收拾东西、最后一次回来、周末带孩子回家。
2. 情绪视觉钩子：用一个可拍到的光线或天气把观众带入情绪，如蓝调、晨光、雨天、晚灯、窗景、阳台风。
3. 生活选择或动作：写人接下来会做什么，如刷剧、做饭、工作、陪孩子、打包、整理、洗漱、发呆。
4. 房源硬信息自然嵌入：面积、户型、楼层、配套、景观、学区、通勤等只作为生活场景的支撑，不集中报参数。
5. 空间动线承接叙事：按真实行走顺序，从公共空间到私密空间，或从功能区走向情绪落点。
6. 轻口语收尾：用一句祝福、提问、告别或交接结束，不用广告式总结。

五类故事线适配方式：

- 局部亮点型：用日常麻烦开场，再进入解决它的小空间。
- 装修省心型：用下班、搬家第一周或入住第一晚开场，再讲省掉的折腾。
- 业主个人叙述型：用最后一次记录或最后一次回来开场，再讲记忆、打包和交接。
- 看房人纠结型：用真实生活需求开场，再讲为什么心动以及卡在哪里。
- 搞笑抽象互动型：用夸张生活设定开场，再把每个空间变成设定里的爽点。

使用边界：

- 不要把在住口吻直接套到告别、看房或搞笑故事里。
- 不要复用样本中的具体事实、娱乐设备、地段、面积或原句。
- 如果当前房源没有窗景、光线、社区或大面积卖点，就换成可确认的小动作和真实空间细节。

## 故事线选择

### 1. 局部亮点型

客观中立口吻，适合不起眼但解决真实痛点的空间或设计，如储物间、家政区、玄关柜、双台盆，可以从人工输入的亮点信息中提取1个作为重点讲解。

结构：

1. 用“小东西胜过大卖点”的反差开场。
2. 写原本以为的核心看点。
3. 揭示真正留下叙述者的局部亮点。
4. 列举 3—5 个普通生活物品或麻烦。
5. 说明它如何让其他空间恢复原本功能。
6. 交代结构、承重、通风、防潮或使用边界。
7. 用“对这个户型来说，它到底加不加分”等反问对比式语调收口。

常用句式：

- “最打动我的，居然是……”
- “这些东西在户型图上从来不出现，但……”
- “它很边缘，却决定了……”
- “当然，实际能不能……还要现场确认。”

### 2. 装修省心型

适合状态完整、短期不用重装、适合赶时间入住的房源。

结构：

1. 从看二手房的共同疲惫切入。
2. 点出“买房以后还要继续过工地生活”的痛点。
3. 说明这套不是简单摆拍，而是生活系统已经成形。
4. 描写搬家第一周或下班回家的具体场景。
5. 把价值落到时间、精力和情绪成本。
6. 提醒风格偏好、装修折价与房价仍需拆开判断。
7. 询问“省掉这段折腾值多少钱”。

常用句式：

- “我现在最怕的不是旧，是……”
- “进去第一反应不是入住，是开工。”
- “更具体一点，是搬家后的第一周。”
- “安家不一定非要从灰尘里开始。”

### 3. 业主个人叙述型

适合搬家、离开城市、家庭阶段变化或出售自住房，是以前业主的身份而并非刚搬入小区的人。

结构：

1. 先说明为什么要告别。
2. 回到买房、装修或两个人生活的起点。
3. 沿空间顺序讲 4—6 个带人物关系的生活片段。
4. 穿插装修选择及其真实使用方式。
5. 用已经清空、打包或最后一次观看制造离别感。
6. 把房子交给下一任主人，并将人生推向下一阶段。

写法要求：

- 每个空间至少绑定一个动作或记忆，不单纯介绍配置。
- 情绪递进要克制，从记录、回忆到舍不得，最后接受离开。
- 避免连续煽情，使用家具、第一顿饭、收走的收藏等具体物件托住情绪。

常用句式：

- “走之前，最后再好好记录一下……”
- “这个……也是我们一起……的。”
- “还记得入住第一餐……”
- “行李已经打包好了。”

结尾语气：
- “再见了……希望……”
- “……新的旅程……”

### 4. 看房人纠结型

适合优点明确但存在长期使用矛盾、预算冲突或改造代价的房源。

纠结点提取：
- 当卫生间数量等于1，卧室大于2时，卫生间使用压力成为一个纠结点。
- 当改造涉及到将原有卧室改为其他功能房间时成为一个纠结点，具体要看改造后的房间类型，如果是补充了之前户型没有的客厅、餐厅，可以说满意，但是确实也压缩了居住空间，毕竟可居住卧室减少了。


结构：

1. 开门见山说明“最近看了很多……有点心动，也有点纠结”。
2. 交代家庭阶段和基本需求。
3. 用 2—3 组事实说明为什么心动。
4. 用“但问题来了”抛出唯一核心顾虑。
5. 展开现在舒服与未来需求之间的冲突。
6. 与其他备选房源的代价形成对照。
7. 请求有相似经验的人给出选择建议。

写法要求：

- 优点与顾虑都要具体，不替观众下结论。
- 顾虑必须影响真实决策，如孩子、工期、预算、恢复成本或收纳。
- 结尾明确两种选项，例如“接受以后再改”或“继续找状态一般但格局原始的房子”。

常用句式：

- “对这套有点心动，又有点纠结。”
- “我们现在这样住当然舒服，但……”
- “可看过的其他几套……”
- “所以现在就卡在这里。”

### 5. 搞笑抽象互动型

适合轻娱乐、闺蜜幻想、夸张代入或评论区传播。

结构：

1. 用艾特、许愿或夸张身份快速拉互动。
2. 建立一个统一幻想设定，并贯穿全文，例如闺蜜，姐妹，兄弟，朋友，总裁等，根据身份转换语气。
3. 每到一个空间，都把普通配置升级成生活爽点。
4. 使用网络口语和短促感叹，但不过度堆梗。
5. 在结尾把设定再翻一次，形成二次笑点。

写法要求：

- 夸张的是想象，不得夸大房源事实。
- 每个空间的开场白都是核心围绕着幻想设定主角展开，幻想主角在里面具体的生活。
- 每 3—5 句最多出现一个强网络词，避免“梗比房子多”。
- 不拿真实群体、经济处境或居住缺陷做攻击性笑料。
- 幻想主角在里面生活时，以“你闺蜜”、“你朋友”、“总裁你的……”类似强调身份的句式作为开头。

常用句式：

- “赶紧艾特你那个……”
- “往沙发上一瘫就是一天。”
- “不存在的。”
- “所以问题来了——”

结尾语气：
- “你是艾特你那个……给你买这个，还是买……”
- “听说评论区艾特……就……”

## 通用骨架

```text
[一句反差、处境或互动钩子]
[补充房源基本信息，但不集中报参数]

[原本预期]
[真正吸引人的地方]
[生活动作或具体物件]
[它解决的麻烦]

[第二个生活场景]
[把空间功能转成时间、情绪或关系价值]

[真实限制或长期顾虑]
[为什么仍然值得讨论]

[与核心矛盾直接相关的互动问题]
```

## 改写原则

- 学习节奏和结构，不重复示例中的整句表达。
- 同一账号连续发布时，轮换钩子句式和转折词。
- 同一卖点至少准备两种角度：解决麻烦、创造场景。
- 事实不够支撑故事时，缩短脚本，不用虚构经历填满时长。
